# Software installed before start using OC

The following software must be installed in the computer before start using OC:

- Beckhoff TwinCAT 3 Engineering XAE version 4026 (x64)
- Beckhoff Package Manager
- Unity Hub (latest version)
- Unity Editor (version 2022.3 or later)

Even though there are examples within the documentation of OC, it is recommended that the user already has a previous knowledge of these software tools.

# Installation of the OC basic environment

To start creating simulations with the OC platform it is required to install the following components, which are the base of the Open Commissioning platform.

1. OC_Unity_Core
2. OC_TwinCAT_Core
3. OC_Assistant

All these components can be found in the GitHub repository of OC under the following link:

[Open Commissioning](https://github.com/OpenCommissioning)

**NOTE:** It is always recommended to use the released versions of the different components of OC for creating simulations.
