
## Enable play mode running in the background

To enable the scene run in play mode when the mouse is interacting with other applications.

1. go to *Edit*
2. select *Project settings*
3. select the option *Player*
4. Check the box  *Run in Background*
5. close the project settings window

![[Enable_running_in_background.png]]
## Change the handling position to the origin of the game object

In the upper left corner of the *Scene* tab click in the drop down and select **PIVOT** 

![[Activate pivot display.png]]
## Create new game objects in the origin of the scene

1. go to *Edit*
2. select *Preferences*
3. select the option *Scene view*
4. Look for the property *3D placement mode*
5. Select **World Origin**
6. close the preferences window

![[Project_preference_settings.png]]

## Enable constrained proportions

![[Enable_constrained_proportions.png]]

By editing the size properties of a game object, it is possible to enable *constrained proportions* to modify the size of the object in the 3 axes at the same time in the same ratio.
