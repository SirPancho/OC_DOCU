
![[select_the_offset_mesh.png]]

1. Drag the prefab into the scene

![[drag_prefab_into_main_scene.png]]

2. Manually position the prefab object as close as possible to the mesh to be replaced

![[rough_position_of_prefab_with_offset_mesh.png]]

3. With the *prefab* selected click the key #V_click  to enable the vortex selection and select a node of the prefab's mesh, preferably where are not so many connections and click #right mouse button.
4. Drag the *prefab* to match the same node in the *offset mesh*. basically until both meshes "blend" with each other.
**NOTE:** Make use of the surface display mode and the isometric views to fit the meshes as close as possible.

![[overlap_prefab_mesh_with_offset_mesh.png]]

5. Compare the position of both meshes by hiding the mesh of the prefab #H_click 
6. When the *prefab* is correctly placed, then it is possible to delete the mesh with offset from the scene
