



## Enable/Disable of elements 

It is possible to enable or disable game objects without deleting them from the #Scene or the #prefab only by unmarking the check box next to the *name* of the game object, it will be disabled from the Scene hierarchy, visually will be hidden and all the scripts will not be executed.

![[Pasted image 20260724155651.png]]

![[Pasted image 20260724155742.png]]

This can be useful for the prefab concept, because inside the #prefab can be defined several objects wich are not going to be used in all the instances of the same prefab, but they will be the for enabling if required.
